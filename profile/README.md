# 프로젝트 개요

이 프로젝트는 리뷰 품질 개선을 통한 마케팅도우미 "리뷰 할래?" 서비스입니다.   
이 서비스는 파인튜닝된 AI를 사용해 소비자들의 리뷰 작성을 촉진시키고, 리뷰 품질을 개선하여 기업의 마케팅을 도와주는 B2B 서비스입니다.    
소비자의 리뷰 작성 시간 절약과 리뷰 품질 향상에 중점을 두며, 높은 품질의 리뷰가 많아짐에 따라 SEO와 마케팅 효과를 극대화할 수 있습니다.
