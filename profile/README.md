# 프로젝트 개요

이 프로젝트는 리뷰 품질 개선을 통한 마케팅도우미 "리뷰 할래?" 서비스입니다. 사용자들이 공모전 정보를 확인하고 지원하며, 참가자 리스트를 관리하고 계약서를 작성할 수 있는 기능을 제공합니다.
